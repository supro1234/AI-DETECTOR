Installation Guide

Standard deployment workflow for modern workstation environments.

1. Machine Preparation

Ensure Node.js (v18+) and Python (3.10+) are installed. If you face execution errors on Windows PowerShell, bypass terminal policy:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

2. Dependency Provisioning

# Root folder (Installs concurrently for Node)
npm install

# Backend folder (Python dependencies)
pip install google-generativeai groq requests

3. Neural Launch

Start the forensic lab in dev mode. Concurrently boots both Express and Vite servers.

npm run dev

Forensic Methodology

Anatomical Logic

AI generators often hallucinate extra digits or skewed joints. The engine inspects finger articulation, pupillary symmetry, and ear placement consistency.

Lighting Physics

Detects "impossible shadows" that are detached from their source or cast in directions conflicting with the primary light source.

Background Artifacts

Identifies 'ghosting' (half-formed objects) and repetitive patterns in organic textures like clouds, grass, or water.

Edge Consistency

Zooms in on fine details like hair strands. Detects the 'waxy/plastic' sheen or unnatural sharpness characteristic of AI upscaling and smoothing.

Neural API Matrix

Unified interface for cross-model verification.

Troubleshooting

  • ECONNREFUSED: Ensure your backend is running on localhost:8301. Check terminal for Node.js errors.
  • API Auth Failed: Tokens are case-sensitive. Verify provider selection (Gemini vs Groq vs OpenRouter).
  • Python Not Found: If python fails, update server.js line 14 to 'python3' for macOS/Linux users.
  • Upload Limit: Maximum file size is 50MB. High-res TIFFs should be converted to JPG/PNG for faster analysis.
